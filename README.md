## 📌 Description
Browser extension for automating interactions with dynamic web interfaces using DOM manipulation and scripted logic.

This project demonstrates building scenario-based automation with state management, asynchronous processes, and interaction with changing UI elements.

---

## ⚙️ Features
- automation of repetitive user actions  
- DOM interaction (element selection, event handling)  
- scenario-based logic with conditions and loops  
- state management for different automation modes  
- asynchronous behavior using async/await  
- simple UI for controlling automation (start/stop)

---

## 🛠 Tech Stack
- JavaScript  
- Chrome Extensions API (Manifest V3)  
- DOM API  

---

## 💡 Implementation Details
- configurable selectors and text markers  
- handling dynamic page updates  
- event-based automation  
- separation of logic and configuration  

---

## ⚠️ Note
This project is designed for demonstration purposes.

To use it in a real environment, selectors and text markers must be adapted for the specific web interface.

## 🚀 Getting Started
1. Clone the repository  
2. Open Chrome → Extensions  
3. Enable Developer Mode  
4. Load unpacked extension  
5. Configure selectors inside the script
