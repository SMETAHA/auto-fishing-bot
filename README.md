# Web Automation Extension

## 📌 Description
Browser extension for automating interactions with dynamic web interfaces using DOM manipulation and scripted logic.

## ⚙️ Features
- automation of repetitive user actions
- DOM interaction (element selection, events handling)
- scenario-based logic with conditions and loops
- simple UI for controlling automation (start/stop)

## 🛠 Tech Stack
- JavaScript
- Chrome Extensions API (Manifest V3)
- DOM API

## 💡 Implementation Details
- state management for different automation modes
- asynchronous logic using async/await
- interaction with dynamic page elements
- configurable selectors and behavior

## ⚠️ Note
To use this extension, selectors and text markers must be adapted for the target web interface.
