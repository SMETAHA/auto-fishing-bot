# Web Automation Extension

🇷🇺 Русский | [🇺🇸 English](README.md)

---

## 📌 Описание
Браузерное расширение для автоматизации действий в веб-интерфейсах с использованием DOM и сценарной логики.

Проект демонстрирует построение автоматизации с управлением состояниями, асинхронной логикой и взаимодействием с динамическим интерфейсом.

---

## ⚙️ Возможности
- автоматизация повторяющихся действий  
- работа с DOM (поиск элементов, обработка событий)  
- сценарная логика (условия, циклы)  
- управление состояниями  
- асинхронная логика (async/await)  
- простой UI для управления  

---

## 🛠 Стек
- JavaScript  
- Chrome Extensions API (Manifest V3)  
- DOM API  

---

## 💡 Особенности реализации
- настраиваемые селекторы и текстовые маркеры  
- обработка динамического интерфейса  
- событийная логика  
- разделение конфигурации и логики  

---

## ⚠️ Важно
Проект представлен в демонстрационных целях.

Для использования необходимо адаптировать селекторы и текстовые маркеры под конкретный интерфейс.

## 📂 Структура проекта

manifest.json

---

## 🚀 Запуск
1. Скачать репозиторий  
2. Открыть Chrome → Расширения  
3. Включить режим разработчика  
4. Загрузить распакованное расширение  
5. Настроить селекторы
